# ABSON Real Estate Project 🚀

Egy modern, Spring Boot alapú ingatlankezelő rendszer.

## Funkciók
- **REST API**: Teljeskörű ingatlankezelés.
- **Swagger UI**: Interaktív API dokumentáció a `/swagger-ui.html` alatt.
- **Security**: Admin felület védelem (Basic Auth).
- **CI/CD**: Automatikus telepítés GitHub Actions-szel.
- **Docker**: Konténerizált futtatás PostgreSQL adatbázissal.

## Éles link
[https://abson.dev](https://abson.dev)

## Tech Stack
- Java 21, Spring Boot 3.x
- PostgreSQL, Docker & Docker Compose
- GitHub Actions, Cloudflare, DigitalOcean
